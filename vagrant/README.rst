Test using virtual machine.


Install:

 * install vagrant_
 * install VirtualBox_
 
Run::

    # to start box
    cd vagrant
    vagrant up
    # login
    ssh -X vagrant@10.0.0.100
    password:vagrant
    # start any demo
    python -m pysimavrgui.examples.sim.lcd


.. _vagrant: http://www.vagrantup.com/
.. _virtualbox: https://www.virtualbox.org/
